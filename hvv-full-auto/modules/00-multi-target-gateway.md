# 模块 00：多目标入口 / 分诊网关

适用：
- 目标数量多于 20
- 存在资产清单、Excel、CSV、CIDR、批量 URL
- 用户要求“先筛选好打目标”

核心原则：
1. 先归一化、去重、聚类
2. 先用 nuclei 识别层，再映射 N-day
3. 先看价值，再看可利用性
4. 先挑样本，再扩同簇
5. 先定向批量锁定，再回流单目标深打
6. fallback 轻探针只补充 robots/login/favicon 等字段，不替代 nuclei 识别

不要：
- 一上来对全量目标跑重型漏洞利用
- 不分行业/系统价值，机械地逐个深打
- 不聚类，重复分析同指纹目标
- 不看识别结果就全库跑模板
