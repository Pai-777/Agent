# Unpacking Skills 

## 项目简介

这是一个集成的**压缩壳**检测与脱壳工具集，专门用于检测和移除二进制可执行文件中的加壳保护，提供自动化的两步式脱壳流程。

## 功能特性

- **智能检测**：使用DIE工具识别多种常见壳类型
- **版本提取**：自动提取加壳工具的具体版本号
- **自动脱壳**：调用对应工具进行脱壳处理
- **安全备份**：自动创建带时间戳的原文件备份
- **灵活输出**：支持覆盖原文件或指定输出路径
- **详细日志**：提供完整的检测和脱壳执行日志
- **退出码管理**：清晰的退出码便于自动化脚本调用

## 目录结构

```
unpacking-skills1/
├── README.md                           # 项目说明文档
└── skills/
    └── unpacking-skills/               # 脱壳工具主目录
        ├── Detect.py                   # 壳检测脚本
        ├── Unpack.py                   # 自动脱壳脚本
        ├── SKILL.md                    # 技能详细说明
        └── scripts/                    # 工具可执行文件目录
            ├── diec/                   # Detect It Easy 工具
            │   ├── diec.exe            # DIE主程序
            │   └── db/                 # 特征数据库
            └── upx/                    # UPX 脱壳工具
                └── upx.exe             # UPX官方工具
```

## 支持的壳类型

当前版本支持检测以下常见加壳工具：

| 壳类型 | 检测支持 | 脱壳支持 |
|--------|---------|---------|
| **UPX** | ✅ | ✅ |
| **ASPack** | ✅ | ⏳ |
| **PECompact** | ✅ | ⏳ |
| **RLPack** | ✅ | ⏳ |
| **NSPack** | ✅ | ⏳ |
