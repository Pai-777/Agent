# 优先级评分模型

## 总分

```text
总分 = 资产价值 * 0.35
     + 可利用性 * 0.35
     + 证据信号 * 0.20
     - 操作成本 * 0.10
```

## 资产价值

高分：
- SSO / CAS / 4A
- OA / 邮件 / VPN / 堡垒机
- K8s / 云控台 / 运维平台
- 教务 / 一卡通 / 财务
- HIS / PACS / LIS / EMR
- 数据库管理 / 中间件控制台

## 可利用性

高分：
- 命中本地 POC / nuclei 模板
- 老 CMS / 老框架 / 明显版本
- 上传 / 后台 / API / 调试口
- SQL 报错、路径泄露、信息泄露

## 证据信号

加分：
- `robots.txt` 泄露路径
- JS 泄露接口
- 标题暴露产品
- header / cookie 暴露组件
- favicon 明显命中产品

## 操作成本

扣分：
- 强 WAF / 验证码 / 频繁 403
- 超慢 / 不稳定
- 同簇重复目标太多
- 需要复杂交互而价值一般
